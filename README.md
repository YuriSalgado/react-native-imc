## Installation

Use the package manager [npm](https://www.npmjs.com/) and platform [expo](http://expo.io/) to start app.

```gitbash
npm install --global expo-cli
```

## Usage

```gitbash
expo start
```
 Open in your browser > http://localhost:19002 > choose run as..
